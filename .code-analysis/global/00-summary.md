## 1. 项目概览
Velpos 是一个通过 Agent SDK 控制 Claude Code 的 Web 界面系统，采用 **Python FastAPI 后端 + Vue 3 前端** 的前后分离架构，支持多 IM 渠道（飞书、OpenIM、QQ、微信）集成。
---
## 2. 核心发现
### 2.1 架构设计优秀
- **DDD 四层架构**清晰分离：`domain`（纯业务逻辑）→ `application`（用例编排）→ `infr`（技术实现）→ `ohs`（对外接口）
- **防腐层(ACL)** 模式应用恰当，领域层通过接口定义与外部依赖解耦
- **插件式 IM 适配器**通过注册表模式支持多渠道扩展，新增渠道仅需实现适配器
### 2.2 技术栈现代化
- 全链路异步设计：**FastAPI + SQLAlchemy async + aiomysql**，性能优秀
- **uv** (Python) + **Vite** (JS) 是各自领域的一流工具链
- 容器化完善，支持 dev/prod 双环境部署
### 2.3 缺少测试保障
- 前后端均无测试目录
- 无 pytest / Vitest 配置
- **这是最大风险点**，代码可靠性无法自动验证
### 2.4 代码检查工具缺失
- 无 ESLint / Prettier 配置
- 无 Pylint / Black / Ruff 配置
- 代码风格一致性依赖人工Review
### 2.5 部分文件职责过重
- `main.py` (350+ 行) 承担过多职责
- `App.vue` (550+ 行) 组件过大
